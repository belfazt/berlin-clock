# Berlin Clock

## Purpose
Represent a [Berlin Clock](https://en.wikipedia.org/wiki/Mengenlehreuhr) within a JVM language

## Building
```bash
$ ./gradlew assemble
# Starting a Gradle Daemon, 1 stopped Daemon could not be reused, use --status for details
# :compileJava
# :processResources NO-SOURCE
# :classes
# :jar
# :assemble

# BUILD SUCCESSFUL

```
